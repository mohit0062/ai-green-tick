# Shadcn Studio Export

This project was exported from [Shadcn Studio](https://shadcnstudio.com).

## Getting Started

1. Install dependencies:

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

2. Install Shadcn components:

    ```bash
    npx shadcn@latest add badge button card avatar utils
    ```

3. *(Optional)* Install your theme by following the instructions on [Shadcn Studio](https://shadcnstudio.com).

4. Run the development server:

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
